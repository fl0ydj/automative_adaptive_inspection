\doxysubsection{Detailed Description}
Real Sense Wrapper class. 

This class allows to create pointclouds, remove the background in a pointclouds, perform the R\+GB intrinsic and extrinsic calibration. Sources\+: \href{https://docs.opencv.org/3.4.9/df/d4a/tutorial_charuco_detection.html}{\texttt{ https\+://docs.\+opencv.\+org/3.\+4.\+9/df/d4a/tutorial\+\_\+charuco\+\_\+detection.\+html}}, \href{https://github.com/IntelRealSense/librealsense/issues/1601}{\texttt{ https\+://github.\+com/\+Intel\+Real\+Sense/librealsense/issues/1601}}, \href{https://github.com/IntelRealSense/librealsense/issues/1021}{\texttt{ https\+://github.\+com/\+Intel\+Real\+Sense/librealsense/issues/1021}} \begin{DoxyAuthor}{Author}
Justin Heinz 
\end{DoxyAuthor}
