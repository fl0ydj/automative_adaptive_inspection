Detailed Description

Class for the subproblem of the registration.

This class allows postprocessing functionalities, estimation of normals, downsampling and performing the registration using SAC Initial Alignment or ICP. Source: http://pointclouds.org/documentation/tutorials/template_alignment.php

Author
Justin Heinz
