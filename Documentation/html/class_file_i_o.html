template<typename T >
bool FileIO::loadCloud ( std::string  text,
typename pcl::PointCloud< T >::Ptr  cloud,
std::string  def = "",
bool  scalingRequired = false 
)

Query a user for a path to a pointcloud and load it. Supported file formats: .pcd, .stl, .ply, .txt, .jt For .txt files, loadResults(path) is called. For .jt files, loadTolerances(path) is called.

Parameters
[in]text- Text to show to the user to ask for a path.
[out]cloud- the cloud loaded as a response
[in]def- When set, it is the default path name to use, when the user types "default".
[in]scalingRequired- When set, queryScaling is called.
Returns
true when the cloud was loaded successfully, false when an error occured

Query the user to enter the unit which is used for the cloud. The cloud is scaled accordingly. Supported: mikro,mm,cm,dm,m.

Parameters

[in,out]cloud- The pointer to the scaled cloud. If empty, tolerance is scaled instead.
Returns
true when successful, false when an error occured or an invalid unit was entered.
